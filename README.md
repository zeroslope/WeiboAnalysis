## How to run

### install node.js

请确保爬虫项目与当前项目路径为如下关系：
```
.
|__ electron
|__ weibo_scrapy
```

* 安装nvm，参考 [Node Version Manager](https://github.com/creationix/nvm)
* 使用nvm安装node.js: nvm install 10.7.0

```shell
npm install -g cnpm --registry=https://registry.npm.taobao.org
# 在该目录下
cnpm install
cnpm run start
```
